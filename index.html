D3 - Progress indicators

Example 1: Visually highlighting how complete an item is, overrides some simple display options. Additional styling is done via css. 

    new ProgressIndicator({
      selector: "#progress-indicator-1",
      radius:50,
      fontSize:35
    }, {
      percentageCompleted: 92,
    });

Example 2: Inclusion of milestone data, colour indication of which milestones have been achieved. 

    new ProgressIndicator({
      selector: "#progress-indicator-2",
      xPadding: 110
    }, {
      percentageCompleted: 78,
      milestones: [{
        value: 25,
        text: "Account created"
      }, {
        value: 50,
        text: "Profile complete"
      }, {
        value:75,
        text: "Profile visible"
      }]
    });

Example 3: Another milestones example with a few additional display overrides 

    new ProgressIndicator({
      selector: "#progress-indicator-3",
      radius:60,
      fontSize:40,
      milestoneTriangle:5,
      xPadding: 50
    }, {
      percentageCompleted: 27,
      milestones: [{
        value: 20,
        text: "Level 1"
      }, {
        value: 40,
        text: "Level 2"
      }, {
        value: 60,
        text: "Level 3"
      }, {
        value: 80,
        text: "Level 4"
      }]
    });
